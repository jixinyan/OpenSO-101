# Constraints applied to BOTH the install target and pip's isolated
# build environments. Use via:
#
#   PIP_CONSTRAINT=$(pwd)/constraints.txt pip install -e . -v
#
# Why each entry:
#
# setuptools<81
#   setuptools 81 removed `pkg_resources` from the default install
#   (https://github.com/pypa/setuptools/issues/4961). Some transitive
#   sdist deps — most notably `flatdict` 4.0.1, pulled in through
#   lerobot's dependency chain — still `import pkg_resources` from
#   their legacy setup.py. Without this pin, `pip install -e .` fails
#   with "ModuleNotFoundError: No module named 'pkg_resources'" during
#   flatdict's wheel build. Holding setuptools below 81 keeps the legacy
#   shim available in pip's isolated build env so those sdists install.
setuptools<81
